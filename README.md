# Wekeza Capstone Project

Quantitative Investment Platform

This repository contains the landing page for the Wekeza Capstone Project, a platform offering portfolio management and investment options using quantitative analysis and AI agents.

## Repository Information
- **Repository Name**: `wekeza`
- **Description**: Webpage for the Wekeza Capstone Project.

## Branch Structure
- **`main`**: The primary branch containing the production-ready code for the website.
- **`feature/*`**: Feature branches for developing new sections or styling (e.g., `feature/team-bios`, `feature/dark-mode`).
- **`hotfix/*`**: Emergency fixes for the production site.
